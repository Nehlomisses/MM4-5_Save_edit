魔法門外傳合輯修改器 (MM4&5 Save Edit)<br>
此修改器僅限魔法門外傳1+2合輯專用 如果只單一安裝魔法門外傳1或2 無法使用<br>
官方已將魔法門外傳正名為魔法門4與魔法門5了<p>

使用說明：<br>
預設路徑是與檔案同資料夾的路徑，建議把修改檔放到遊戲目錄內(官方預設為C:\XEEN)，方便快速讀取<br>
每次只能修改一個隊員，修改後按存檔才能點下一個隊員繼續修改，否則會只有修改到最後一個選擇到的隊員<br>
EX：隊員一修改完後按存檔=>繼續修改隊員2並按存檔=>隊員1、2都有修改到<br>
    隊員1修改完後直接點隊員2=>修改隊員2後才按存檔=>只有修改到隊員2<p>
